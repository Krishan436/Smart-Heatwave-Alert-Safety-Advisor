# 🌡️ Smart Heatwave Alert & Safety Advisor

An AI-powered web application that provides real-time heatwave risk alerts, weather monitoring, heat index analysis, personalized safety recommendations, and emergency guidance to help users stay safe during extreme temperatures.

---

## 🚀 Features

- 🌡️ Real-time weather monitoring
- 📍 Location-based weather search
- 🚨 Heatwave risk alerts
- 📊 Interactive Heat Index gauge
- 💧 Personalized hydration recommendations
- 👨‍👩‍👧 Safety profiles for children, elderly, outdoor workers, pregnant women, and pets
- 📈 Weather forecast visualization
- 🧮 Heat Index calculator
- 🎮 Heatwave scenario simulator
- 📱 Responsive modern dashboard

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Lucide Icons
- Open-Meteo API
- Browser Geolocation API
